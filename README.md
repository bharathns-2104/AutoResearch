# AutoResearch

AutoResearch is an automated business idea research and report generation system.

## Features
- Market analysis
- Financial analysis
- Competitive analysis
- PDF and PPT report generation

## Tech Stack
- Python 3.9+
- Streamlit
- DuckDuckGo Search
- BeautifulSoup
- ReportLab

## Project Structure
- src/ → core source code
- config/ → configuration files
- data/ → cache and outputs
- logs/ → logs
- tests/ → unit tests
